JSON-to-NestJS

Convert JSON to NestJS DTO

This tool instantly converts JSON into a NestJS DTO definition. Paste a JSON structure on the left and the equivalent DTO will be generated to the right, which you can paste into your program. The script has to make some assumptions, so double-check the output!
